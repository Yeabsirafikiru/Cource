Lesson Overview

This lesson introduction the css grid layout system. you'll learn key concepts, view examples, and your knowledge with a short queiz.

Estimate Time: 20-30 minutes. Difficulty: Medium

Learning Objectives

  1. Understand the basic terminology of css grid (container, tracker, cell, gap.)
  2. Create a simple two-dimentional layout using grid-template-columns and grid-tenplate-rows.
  3. Place items with grid-column and grid-row, and use shorthand grid-area.

Example: Simple Grid markup

<div class="grid">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>

And CSS such as:

.grid {
        display: grid;
        grid-template-columns: 1fr 2fr;
        gap: 16px;
      }
